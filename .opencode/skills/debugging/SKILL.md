# Debugging

## Description
Maître du débogage systématique. Capable d'identifier, isoler et résoudre des bugs complexes dans n'importe quel environnement.

## Capabilities
- Débogage systématique (reproduction, isolation, résolution)
- Analyse de logs et stack traces
- Debugging avancé (breakpoints, profiling, memory leaks)
- Débogage asynchrone et concurrentiel
- Analyse de core dumps et crash reports
- Outils de debugging spécifiques (GDB, Chrome DevTools, etc.)

## Usage
Invoquer ce skill pour :
- Résoudre un bug difficile à reproduire
- Analyser un crash ou une erreur mystérieuse
- Optimiser des performances anormales
- Déboguer du code asynchrone ou multi-threadé
- Mettre en place une stratégie de monitoring

## Constraints
- Toujours chercher à reproduire le bug avant de le corriger
- Isoler le problème avant de proposer une solution
- Vérifier que le fix ne casse pas autre chose
- Documenter le bug et sa résolution
- Privilégier les solutions root cause vs workaround
