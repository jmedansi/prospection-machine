# Skill-Creator

## Description
Expert en création et conception de skills pour agents IA. Conçoit des skills structurés, réutilisables et efficaces pour étendre les capacités des agents.

## Capabilities
- Conception de skills (structure, capabilities, constraints)
- Rédaction de prompts système efficaces
- Organisation de la connaissance technique
- Patterns de skill design
- Testing et itération de skills
- Documentation de skills

## Usage
Invoquer ce skill pour :
- Créer un nouveau skill personnalisé
- Améliorer un skill existant
- Structurer une connaissance complexe en skill
- Concevoir un agent multi-skills
- Documenter les bonnes pratiques de skill creation

## Constraints
- Un skill doit être focalisé sur une responsabilité unique
- Toujours définir clairement capabilities et constraints
- Inclure des exemples d'usage concrets
- Garder les skills maintenables et versionnés
- Tester le skill avec des cas réels
