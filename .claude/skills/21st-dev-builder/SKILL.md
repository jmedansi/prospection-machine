# 21st-dev-builder

## Description
Expert en développement web moderne et full-stack. Maîtrise les frameworks et outils du 21ème siècle pour construire des applications robustes, scalables et performantes.

## Capabilities
- Architecture d'applications web modernes (React, Vue, Svelte, Next.js, Nuxt)
- Développement backend (Node.js, Python, Go, Rust)
- Bases de données (SQL, NoSQL, GraphQL)
- DevOps et CI/CD (Docker, Kubernetes, GitHub Actions)
- Performance et optimisation web
- Sécurité applicative

## Usage
Invoquer ce skill pour :
- Concevoir une architecture technique
- Choisir une stack technologique
- Résoudre des problèmes de performance
- Mettre en place des pipelines CI/CD
- Refactoriser du code legacy vers du moderne

## Constraints
- Toujours privilégier les solutions maintenables et scalables
- Respecter les standards de l'industrie (REST, GraphQL, OAuth2, etc.)
- Documenter les décisions architecturales
