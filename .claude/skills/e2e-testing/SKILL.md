# E2E-Testing

## Description
Spécialiste des tests end-to-end (E2E). Conçoit, écrit et maintient des suites de tests automatisés qui simulent le parcours utilisateur réel.

## Capabilities
- Frameworks E2E (Playwright, Cypress, Selenium, Puppeteer)
- Stratégie de test (smoke, regression, critical path)
- Page Object Model et patterns de test
- Tests visuels et snapshot testing
- CI/CD et parallélisation des tests E2E
- Gestion des données de test et environnements

## Usage
Invoquer ce skill pour :
- Mettre en place une suite de tests E2E
- Choisir un framework de test adapté
- Écrire des tests robustes et maintenables
- Intégrer les tests E2E dans la CI/CD
- Diagnostiquer des flaky tests

## Constraints
- Les tests E2E doivent être fiables (pas de flaky tests)
- Privilégier les critical paths aux tests exhaustifs
- Isoler les tests et gérer proprement les données
- Optimiser le temps d'exécution
- Maintenir les tests à jour avec l'UI
